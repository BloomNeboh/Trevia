## 🚀 Getting Started
This site is built with **GitHub Pages** for fast, reliable hosting.

**Deployment steps:**
1. Go to **Repository → Settings → Pages**  
2. Select `main` branch and root folder (or `docs/`)  
3. Save → Your site goes live instantly 🎉
